#### 要点
- 使用Bmob来构建即时通讯服务器
- 使用ImageLoader来加载图片
- 使用photoview来预览图片
- 使用百度地图的SDK来实现分享位置的功能
- 使用开源的XListView来实现下拉刷新的功能
